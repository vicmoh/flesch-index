Program: Flesch Index Caluclator

The Problem
-----------
The following index (reference: Rudolf Flesch, "How to Write Plain English", Barnes & Noble Books, 1979) was invented by Rudof Flesch as a simple tool to guage the legibility of document without using linguistic analysis
1) Count all words in the file.
2) Count all syllables in each word.
3) Count all sentences
4) Calculate the Flesch Index of the paragraph.

About
-----
The purpose of the index is to force aythors to rewrite their text until the index is approriately high enough. This is achieved bu reducing the lenghth of sentences and by removing long words.
